Exemplo de estilização de Texto com CSS

Este texto tem um tamanho de fonte maior usando front-size: 24px.

font-weight: Este texto está em negrico com font-weight: bold

font-style: Este texto está em itálico com font-style: #abc

text-align: Este parágrafo está centralizado horizontalmente.

text-align: right: Este parágrafo está alinhado à direita.

line-height: Esta frase tem espaçamento de linha aumentado para falicitar a leitura
O valor aplicado é line-height: 2(2 vezes o font-size)

letter-spacing: Espaçamento entre letras usando letter-spacing: 4px

word-spacing: Este texto tem espaçamento maior entre palavras usando word-spacing: 20px

text-shadows:Texto com sombras aplicada usando text-shadows:2px, 2px, 4px
